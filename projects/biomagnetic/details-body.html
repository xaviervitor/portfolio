Biomagnetic a First-Person Puzzle game about using magnetism to pull and hold heavy and far away objects to solve small puzzles. The character controller consists of standard object holding mechanics, inspired by the Half-Life 2's pickup physics and the gravity gun, but in a more extreme application. At first, the player can only hold small and close objects to get around the level, but once the player gets the Magnetic Bracelet, the core mechanic opens more possibilities.

Features

  • Hold/throw mechanics: Smooth interaction with physics objects, that can be thrown preserving held momentum;
  • Rigidbody button: Activated with a force threshold, that can be smoothly applied by objects above or the player;
  • Rigidbody door: Lifts with an applied upward force and closes with gravity;
  • Trigger system: Hooks button triggers to doors triggers, streamlining puzzle design;
  • Level design: Linear level with many physics-heavy puzzles;
  • Checkpoint/respawn system: Automatic checkpoint at the start of every puzzle;
  • Lighting: Baked lighting and reflection probes using the Unity Lightmapper.
