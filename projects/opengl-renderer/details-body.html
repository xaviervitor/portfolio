This project is a simple graphics renderer, which I started developing to learn more about graphics programming and the C++ language, using the LearnOpenGL book by JoeyDeVries. This project uses glad as the GL loader, GLFW for context, windows, input and events, assimp for model loading and stb for image loading.

Features

  • Phong shading: Dynamic directional lights, point lights, spotlights and ambient lighting;
  • Shaders: Abstractions for reading, compiling and running shaders;
  • Models: Abstractions for importing and rendering textured models;
  • Camera controls: WASD camera navigation.
